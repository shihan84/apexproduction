# Movie Upload Guide

## Artwork

| Field | Size | Ratio | Format | Limit |
|---|---:|---:|---|---:|
| Thumbnail | 1280 x 720 px | 16:9 | JPG/PNG | 2 MB |
| Poster | 1000 x 1500 px | 2:3 | JPG/PNG | 2 MB |
| TV Poster | 1920 x 1080 px | 16:9 | JPG/PNG | 2 MB |

## Add a Movie

1. Open **Movies > Add New Movie**.
2. Upload the thumbnail, poster, and TV poster.
3. Enter name and description.
4. Select **YouTube** for **Trailer URL Type** and enter the trailer URL.
5. In **Basic Info**, enter language, genre, content rating, duration, release date, and access type.
6. Actors, directors, countries, and IMDb rating are optional.
7. Open **Quality Info**, select **YouTube** for **Video Upload Type**, and paste the full-movie URL in **Video URL**.
8. Save.

## YouTube URLs

Use a complete public URL:

```text
https://www.youtube.com/watch?v=VIDEO_ID
https://youtu.be/VIDEO_ID
```

Use **Trailer URL** only for the trailer. Use **Quality Info > Video URL** for the full movie.

## Required Fields

- Name and description
- Trailer type and matching trailer source
- Language and at least one genre
- Content rating, duration, and release date
- Video type and matching full-movie source
- Access type

Paid content requires a plan. Pay-per-view content requires price and purchase settings.
